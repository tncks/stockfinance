# Frontend Web Application

This directory contains the frontend web application for the Stock Finance Trading Simulator.

### Key Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

### Setup and Running

1. **Navigate to this directory:**
   ```sh
   cd packages/webapp
   ```

2. **Install dependencies:**
   ```sh
   npm i
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```
